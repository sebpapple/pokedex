# Pokédex

Application statique de Pokédex en HTML, CSS et JavaScript.

## Déploiement sur GitHub Pages

1. Crée un dépôt GitHub.
2. Uploade tous les fichiers de ce dossier à la racine du dépôt.
3. Va dans Settings > Pages.
4. Sélectionne la branche `main` (ou `master`) comme source.
5. Enregistre et attendez la publication.

Votre site sera disponible à l'adresse :
`https://<votre-utilisateur>.github.io/<nom-du-depot>/`

## Fichiers principaux

- `index.html` : page d'accueil
- `pokemon.html` : page de détail
- `styles.css` : styles
- `app.js` : logique de la page d'accueil
- `pokemon.js` : logique de la page détail
